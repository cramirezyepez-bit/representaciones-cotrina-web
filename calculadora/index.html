Importar presupuesto desde Excel

¿Ya tienes el presupuesto armado en Excel, con el formato habitual (cliente, ítems con medidas, descripción y precio)? Cárgalo aquí para generar la cotización en PDF directamente, con el dibujo técnico de cada ítem incluido automáticamente.

El dibujo técnico se genera detectando palabras como "ventana", "mampara", "puerta", "fija" o "corrediza" en la descripción de cada ítem. Si una descripción no las incluye, el ítem se agrega igual al PDF (con su precio y medidas), solo que sin gráfico.

01

Datos del cliente y proyecto

Configurador de sistema

Mampara = baño/ducha · Muro de vidrio = fachada interior de mediana escala · División de oficina = paneles fijos con escuadras, sin marco perimetral completo (phone booth, divisorios, gerencias). Si dudas entre estas tres, usa División de oficina cuando el trabajo sea solo cristal con escuadras sin estructura de aluminio envolvente.

Cada hoja y mecanismo adicional incrementa el costo (herrajes, carriles, sellado).

Apertura Vidrio Ancho (m) Alto (m)

El alto es opcional — si lo dejas vacío, el paño usa el alto general del vano. Complétalo solo cuando ese paño tenga una medida distinta (ej. un panel fijo más bajo junto a una puerta).

Si el área de una unidad es menor a 1 m², se calcula como mínimo 1 m² por unidad.
